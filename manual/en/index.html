1. Introduction

1.1 What is Peapod?

Peapod is a premium short-term rental marketplace connecting guests with unique accommodations worldwide.

Key Features: - Curated listings reviewed by curator team - Direct host communication via real-time messaging - Loyalty rewards program - Secure payments (Apple Pay, Stripe) - 24/7 support - Fast performance (10-100x faster search) - Enterprise security (bank-level encryption, PCI-compliant)

1.2 User Roles

Guest - Search, book, and review accommodations - Browse properties with advanced filters - Book stays with secure payment - Message hosts - Leave reviews and earn loyalty points - Manage trips and bookings

Host - List properties and manage bookings - Create and edit listings - Handle booking requests - Communicate with guests - Track earnings and payouts - Manage team (co-hosts, cleaners)

Co-Host - Manage assigned properties - Edit listings (except pricing) - Message guests (assigned properties only) - View bookings - Update calendar - Report to property owner

Cleaner - Complete cleaning tasks - View assigned cleaning tasks - Mark tasks complete - Upload completion photos - Report issues - Manage availability schedule

Curator - Review and approve listings - Review listing submissions - Approve or reject properties - Request improvements - Monitor quality standards - Communicate with hosts

Admin - Manage platform - User management - Platform configuration - View analytics and reports - Handle support tickets - Monitor platform health

1.3 System Requirements

  • iOS: 17.0 or later
  • Device: iPhone or iPad
  • Internet: Required for all features
  • Location: Optional (improves search)
  • Notifications: Optional (recommended)

2.5 Role Switching

Understanding Roles:

Every user has a primary role (Guest or Host) that determines their default experience: - Primary Guest - Your primary use is booking stays (you may also host occasionally) - Primary Host - Your primary use is listing properties (you may also travel as a guest)

Special Dedicated Roles: - Curator - Dedicated, isolated role for reviewing listings (cannot be combined with Guest/Host) - Admin - Dedicated, isolated role for platform management (cannot be combined with Guest/Host)

When you sign in, you start in your primary role (Guest or Host) by default.

For Users with Multiple Roles:

If you have multiple roles (e.g., Guest + Host, or Host + Co-Host + Cleaner), you can switch between them.

Switching Your Active Role: 1. Tap Profile tab â†’ tap profile photo 2. Tap Switch Role 3. Select role: Guest, Host, Co-Host, Cleaner (if assigned) 4. App reloads with new tab bar

Frequently Asked Questions

Q: Is my payment information secure? A: Yes. We use Stripe (PCI-compliant). Card details encrypted, never stored. Apple Pay uses biometric authentication.

Q: When will I be charged? A: Authorization hold immediately. Charge captured 24h before check-in. Refunds follow cancellation policy.

Q: How do I cancel a booking? A: Trips â†’ Select booking â†’ â€œCancel Bookingâ€. Refund depends on hostâ€™s cancellation policy (Flexible, Moderate, or Strict). See Guest Manual Appendix D for detailed refund percentages and timing.

Q: What if property doesnâ€™t match listing? A: Document with photos, contact host, report to support. May be eligible for refund.

Q: How do I become a host? A: Sign up with host role, create listing, submit for review. Approval within 24-48 hours.

Q: What fees does Peapod charge? A: CN Release 1.0: Guests pay 0% (no service fee). Hosts pay tier-dependent fees: Silver 13%, Gold 12.5%, Platinum 12%, Diamond 11.5%. See Host Manual Appendix J for details.

Q: Can I modify a booking? A: Not currently - must cancel and rebook. Feature coming Q2 2026.

Q: How do loyalty points work? A: Earn 1% of booking value. 100 points = $1. Redeem on future bookings. CN Release 1.0: Points can ONLY be used for booking discounts - no cash withdrawal available. Points expire after 24 months of inactivity.

Q: Instant book vs request to book? A: Instant book confirms immediately. Request requires host approval (up to 24h).
