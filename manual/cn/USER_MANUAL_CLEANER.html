6. 清洁人员指南

6.1 清洁人员概述

作为清洁人员，您可以： - 🧹 查看清洁任务 - 查看所有分配的清洁工作（自动与预订同步） - 📅 管理日程 - 查看即将到来的周转并设置可用性 - ✅ 完成任务 - 标记工作完成并上传照片 - 🧩 逐房间清单 - 逐个房间完成清单以确保准确性 - 📝 报告问题 - 记录损坏或缺失物品 - 💬 与房东消息 - 就房产和任务进行沟通 - 🔔 保持知情 - 获取新任务分配或反馈的通知

主要功能： - ⚙️ 智能自动分配 - 清洁人员根据房产距离、可用性和绩效历史自动接收新清洁任务 - 🔁 与预订日历自动同步 - 客人退房后自动生成清洁任务 - 📊 房东实时查看 - 房东可以在清单项目被勾选时监控任务进度 - ⭐ 房东评分系统 - 房东可以对每次清洁评分以帮助维护质量标准

6.2 清洁人员导航

底部导航（4个标签）： 1. 任务（星星图标）- 查看和管理清洁任务 2. 日历（日历图标）- 查看日程并设置可用性 3. 消息（气泡图标）- 与房产房东沟通 4. 个人资料（人物图标）- 您的个人资料和设置

工作原理： - 点击任意标签切换视图 - 活动标签以粉色高亮 - 消息徽章显示未读数量 - 您只能看到分配给您的房产的任务

6.3 清洁任务仪表板

位置： 主标签 → “任务”

6.3.1 任务概览

您将看到： 三个标签组织您的工作量： - 今天 - 今天到期的任务（立即优先） - 即将 - 未来计划的清洁 - 过去 - 已完成任务历史

智能分配指示器： 每张任务卡片显示分配方式： - 🔄 系统自动分配（智能自动分配） - 👤 房东手动分配

如何导航： 1. 点击”任务”标签（星星图标） 2. 默认选中”今天”标签 3. 滑动或点击标签切换视图 4. 点击任意任务查看详情

6.3.2 任务卡片信息

每个任务显示： - 房产照片和名称 - 房产地址 - 任务日期和时间 - 任务状态（待处理、进行中、已完成、已跳过） - 优先级指示器（如果紧急） - 特殊说明预览 - 实时清单进度（例如 6/10 项完成）- 房东可实时看到

任务状态颜色： - 灰色 - 待处理（未开始） - 蓝色 - 进行中（您已开始） - 绿色 - 已完成（完成并有照片） - 橙色 - 已跳过（标记为无法完成）

按房产筛选： 如果您清洁多个房产，点击筛选图标仅显示特定房产的任务。

6.4 逐房间任务详情

如何打开： 1. 从任务仪表板 2. 点击任意任务卡片 3. 任务详情视图打开

房产信息： - 房产名称和完整地址 - 房东姓名和联系方式 - 入门说明（门锁密码、钥匙位置等） - 停车信息

任务详情： - 退房时间（前一位客人离开） - 计划清洁开始时间 - 预估持续时间 - 房东的特殊说明

逐房间结构：

每个任务分为房间部分： 1. 卧室 2. 浴室 3. 厨房 4. 客厅 5. 室外（如适用）

每个房间部分包括自己的清单、照片上传区域和可选备注。

示例 - 浴室部分：

[ ] 清洁水槽和台面
[ ] 擦洗淋浴/浴缸
[ ] 消毒马桶
[ ] 补充毛巾
[ ] 清空垃圾桶
[添加照片]

示例 - 卧室部分：

[ ] 更换并整理床铺
[ ] 吸尘地板
[ ] 擦拭表面灰尘
[ ] 检查损坏
[ ] 补充用品
[添加照片]

实时进度： 每个房间的清单完成度更新整体任务进度条，房东可实时看到（例如”7/10 项已完成”）。

入门信息： - 门禁密码或钥匙位置 - WiFi 密码（如需要） - 报警密码（如适用） - 紧急联系人

6.5 完成清洁任务

6.5.1 标记任务开始

如何开始： 1. 从仪表板打开任务 2. 点击”开始任务”按钮 3. 状态变为”进行中”（蓝色） 4. 开始时间自动记录

为什么标记开始？ - 房东知道您已到达 - 跟踪您在现场的实际时间 - 更新团队可见的任务状态 - 房东收到清洁已开始的通知

6.5.2 完成清单

清单如何工作： 每个房间清单可以单独完成。点击复选框标记已完成的项目。

  1. 查看每个房间的清洁项目列表
  2. 完成每个项目时点击复选框
  3. 进度同时在本地和房东仪表板更新
  4. 示例：“7/10 已完成”在房东的房产仪表板实时更新
  5. 必须勾选所有项目才能完成

标准清单项目（典型）： - 更换床单并洗涤 - 用干净床单整理床铺 - 清洁浴室（马桶、水槽、淋浴） - 吸尘所有地板 - 拖硬地板 - 清洁厨房（台面、水槽、电器） - 清空所有垃圾桶 - 补充卫生纸、纸巾 - 检查损坏 - 锁好所有门窗

6.5.3 上传完成照片

如何上传： 1. 完成每个房间的清单后 2. 点击每个房间的”添加照片”按钮 3. 拍摄或选择照片： - 卧室（整理好的床铺） - 浴室（干净的水槽/淋浴） - 厨房（干净的台面） - 客厅（已吸尘） - 任何其他房间（室外、洗衣房等） 4. 每个房间上传至少1张照片显示完成的工作 5. 照片带有时间戳并可供房东查看

照片要求（CN Release 1.0）：

⚠️ 强制要求： 完成清洁任务需要每个房间至少1张照片。

房间类型 最少照片 拍什么
卧室 每间1张 整理好的床铺、干净地板、无尘表面
浴室 每间1张 干净马桶、水槽、淋浴/浴缸、补充的毛巾
厨房 1张 干净台面、电器、空水槽
客厅 1张 吸尘/拖过的地板、整齐的家具
室外 1张（如适用） 扫过的露台、干净的家具

为什么需要每个房间的照片？ - 验证每个房间都被清洁（不只是公共区域） - 为房东审核提供证据 - 付款处理所需 - 保护您在争议情况下

照片技巧： - 良好光线 - 自然光最好（白天照片） - 显示整个房间 - 广角以捕捉整体清洁度 - 聚焦关键区域 - 床铺、浴室、厨房台面 - 避免模糊图像 - 保持手机稳定 - 离开前拍照 - 之后无法返回 - 多角度（可选但推荐）- 显示同一房间的不同视角 - 清晰、明亮的空间 - 如需要打开窗帘

6.5.4 标记任务完成

如何完成： 1. 所有清单项目已勾选（所有房间） 2. 照片已上传（最少4张） 3. 点击”标记完成”按钮 4. 任务状态变为绿色”已完成” 5. 完成时间自动记录 6. 房东立即收到通知

接下来发生什么： - 任务移至”过去”标签 - 房东可以审核您的工作和照片 - 您的完成统计更新 - 任务更新清洁人员绩效指标 - 付款/工时记录（如适用）

6.6 报告问题

何时报告： - 发现损坏 - 破损物品、污渍、墙壁破洞 - 缺失物品 - 床单、毛巾、家具、电器 - 需要维护 - 电器故障、漏水、电气问题 - 入门问题 - 密码错误、被锁在外面、钥匙不工作 - 用品短缺 - 没有清洁产品、床单不足

如何报告： 1. 从任务详情视图 2. 点击”报告问题”按钮 3. 选择问题类型： - 损坏 - 缺失物品 - 需要维护 - 入门问题 - 用品请求 4. 添加描述（具体详细） 5. 上传问题照片（强制要求 - 见下文） 6. 点击”提交” 7. 房东立即收到通知

照片证据要求（CN Release 1.0）：

⚠️ 强制要求： 以下问题类型必须提供照片：

问题类型 照片要求 为什么需要
损坏 ✅ 必需（建议2张以上） 客人收费、保险索赔的证据
额外收费请求 ✅ 必需 额外费用的证明
需要维护 ✅ 必需 维修安排的文档
缺失物品 ✅ 必需 物品位置前后的证据
入门问题 可选 有帮助但非强制
用品请求 可选 有帮助但非强制

没有照片的损坏报告和额外收费请求将被拒绝。

问题报告示例：

类型：损坏
描述："客厅地毯靠近沙发处有咖啡渍。
      直径约12英寸。污渍看起来很新鲜。"
照片：2张不同角度显示污渍的照片
提交：发现后立即

重要指南： - ✅ 在标记任务完成前报告问题 - ✅ 拍照作为证据（损坏/额外收费强制） - ✅ 描述详细（位置、大小、严重程度） - ✅ 当天报告（不要延迟） - ✅ 多角度拍摄损坏（清楚显示范围） - ❌ 不要自己尝试维修（除非房东指示） - ❌ 不要移动损坏物品（留待房东检查） - ❌ 不要提交没有照片的损坏报告（将被拒绝）

6.7 房东评分与反馈

工作原理： 审核您上传的照片和清单完成情况后，房东可以： - ⭐ 评分清洁度（1-5星） - ✏️ 添加备注（“注重细节很棒”、“漏了沙发下面”） - 🧹 在完成后48小时内提交评分

清洁人员仪表板视图： - 查看您的平均评分（在个人资料中可见） - 收到的评价总数 - 最近的房东反馈评论 - 评分趋势

对未来分配的影响： - 更高评分 → 更多自动分配 - 持续5星评分 → 优先状态 - 低评分 → 更少分配、可能审核 - 评分影响智能自动分配算法

评分标准： 房东通常根据以下评分： - 清单完成准确性 - 照片质量和完整性 - 准时性（准时到达/完成） - 沟通响应性 - 整体清洁标准

6.8 日程与可用性

位置： 主标签 → “日历”

6.8.1 查看您的日程

您将看到： - 显示分配任务的日历视图 - 日期上标记任务（彩色圆点） - 点击日期查看任务详情 - 月、周或日视图选项

颜色编码： - 绿点 - 已完成任务 - 蓝点 - 即将到来的任务 - 灰点 - 不可用（您阻止的）

日历集成： - 任务与预订日历自动同步（自动同步功能） - 新任务自动出现 - 变更实时更新

6.8.2 设置不可用日期

如何阻止日期： 1. 进入日历标签 2. 点击您不可用的日期 3. 点击”标记不可用” 4. 可选：添加原因（假期、个人日） 5. 日期标记为灰色 6. 系统在自动分配时跳过您

使用场景： - 假期时间 - 个人预约 - 其他承诺 - 病假 - 休息日

自动分配如何工作： 当您标记为不可用时： - 系统不会为这些日期分配新任务 - 现有任务保留（如需要必须由房东重新分配） - 房东看到您的不可用日期 - 您在轮换中自动被跳过 - 清洁人员收到新任务或重新分配的警报

6.9 消息与沟通

位置： 主标签 → “消息”

6.9.1 与房东消息

您可以消息的对象： - 房产房东（您清洁的业主） - 房产联合房东（如已分配）

您不能消息的对象： - 客人（无客人消息访问） - 其他清洁人员 - Peapod 管理员（使用支持）

如何发送消息： 1. 点击消息标签 2. 点击与房东的对话 3. 输入消息 4. 点击发送

每个房产/任务专用聊天： - 每个房产有自己的消息线程 - 房东看到清洁人员状态更新（“6/10 完成 - 进行中”） - 消息历史保留供参考

消息内容： - 确认任务收到 - 询问房产问题 - 报告发现的问题 - 请求用品或入门信息 - 通知迟到 - 确认任务完成

示例消息：

"任务于下午3:45完成。所有项目已清洁。照片已上传。"

"下次清洁需要更多纸巾。当前供应不足。"

"因交通延误15分钟。将于上午11:15到达。"

"门锁密码不工作。请确认密码是否为1234？"

"浴室水龙头漏水。已在应用中附照片报告。"

沟通最佳实践： - ✅ 2小时内回复房东消息 - ✅ 立即通知房东延误 - ✅ 当天报告损坏 - ✅ 说明不清楚时询问 - ✅ 通过消息确认任务完成 - ✅ 保持专业和礼貌 - ✅ 包含相关详情（时间、位置、具体信息）

6.11 清洁人员最佳实践

时间管理

  • ⏰ 准时到达 - 或延误时通知房东
  • ⏱️ 遵循时长指南 - 尊重预估时间窗口
  • 📱 及时标记任务 - 立即开始/完成
  • 🚫 不要赶时间 - 质量比速度更重要
  • 📅 规划路线 - 最小化房产间的行程时间

质量标准

  • ✅ 完成整个清单 - 不要跳过项目
  • 📸 每个房间上传1张照片（强制）- 卧室、浴室、厨房、客厅
  • 📸 上传清晰照片 - 良好光线显示工作质量
  • 🚨 立即报告问题 - 需要当天报告
  • 📷 损坏照片证据 - 额外收费请求必需
  • 🧴 补充用品 - 如房东提供
  • 🔒 安全锁好 - 检查所有门窗
  • 🧹 做额外努力 - 彻底清洁可见区域
  • 🔍 仔细检查工作 - 离开前巡视

沟通

  • 💬 2小时内响应 - 房东消息
  • 📢 通知房东延误 - 尽快
  • 📝 当天报告损坏 - 不要等待
  • ❓ 询问问题 - 说明不清楚时
  • ✅ 确认完成 - 让房东知道您已完成
  • 🗣️ 保持专业 - 礼貌和尊重的语气
  • 📊 提供详情 - 具体信息有帮助

房产护理

  • 🏠 尊重对待房产 - 像对待自己的一样
  • 🚫 不要移动个人物品 - 非必要
  • 📋 遵循房东指示 - 房产特定规则
  • 🔧 主动报告维护需求
  • 🔐 离开时保护房产安全
  • 🧼 使用适当产品 - 不要损坏表面
  • 👀 保持观察 - 在问题变大前注意

6.12 获取帮助

如果您需要帮助： 1. 首先消息房产房东 - 他们最了解房产 2. 检查任务说明 - 查看特殊说明和备注 3. 联系 Peapod 支持 - 应用问题或技术问题 4. 查看过去任务备注 - 从之前的清洁中学习

支持渠道： - 应用内聊天 - 个人资料 → 帮助 → 联系支持 - 邮件 - support@peapodhotels.com - 电话 - （即将推出）

常见问题：

问：我可以看到客人姓名或预订详情吗？ 答：不能，清洁人员无法访问客人信息或预订详情。您只能看到房产信息和清洁日程。

问：我可以更改我的日程或可用性吗？ 答：可以，在日历标签中标记不可用的日期。系统将在这些日期跳过您的自动分配。

问：如果我无法完成任务怎么办？ 答：立即消息房东，然后将任务标记为”已跳过”并附详细原因。不要在不通知的情况下离开未完成的房产。

问：如何请求清洁用品？ 答：通过应用消息房东。包括所需物品的具体信息和当前库存水平。

问：我可以带助手吗？ 答：先与房东确认。大多数房东出于安全和责任原因要求单独清洁。

问：如果门锁密码不工作怎么办？ 答：立即消息房东。不要试图强行进入。等待更新的入门说明。

问：清洁任务如何分配给我？ 答：任务通过智能自动分配分配，基于：房产距离、您的可用性、绩效评分和房产偏好。

问：如何提高我的评分？ 答：彻底完成清单、上传清晰照片、准时到达、主动沟通、立即报告问题。

问：如果我得到低评分会怎样？ 答：房东可以提供反馈。低评分可能减少未来分配。查看反馈并改进提到的方面。

问：我可以拒绝任务吗？ 答：可以，但频繁拒绝会影响您的自动分配优先级。建议提前标记不可用日期。

7. 房东：管理清洁任务

注意： 本部分适用于房产房东和管理者。如果您是清洁人员，请关注上面的第6部分。

5.8 清洁任务管理

主要功能： - 自动任务创建 - 客人退房后生成 - 手动任务创建 - 随时创建深度清洁、维护或特殊活动任务 - 智能分配 - 自动或明确分配给清洁人员 - 逐房间清单 - 跟踪每个房间的清洁进度 - 照片验证 - 需要前后照片 - 绩效跟踪 - 评价清洁人员质量和速度

5.8.1 自动任务创建

当客人预订您的房产时，系统自动检查是否可以安排清洁：

  1. 触发：客人预订确认
  2. 系统检查：此预订后是否有下一个预订？
  3. 计算截止时间：下次入住前2小时（或如果没有下一个预订则退房后24小时）
  4. 智能分配：找到最佳可用清洁人员（见 5.8.3）
  5. 成功时：任务创建，清洁人员分配并通知
  6. 失败时：房东立即收到警报（< 1分钟）

您需要做什么： - 什么都不用！预订时任务完全自动化 - 在团队标签中监控完成状态 - 任务完成后评价清洁人员 - 除非收到警报，否则无需手动干预

您何时会收到警报： - 🚨 立即：没有清洁人员可以分配（全部超负荷或不可用） - 🚨 立即：退房和下次入住之间时间不足（<2小时） - 在这些情况下，您必须手动分配清洁人员或直接联系清洁人员

  • 任务在预订时创建，而不是退房时
  • 适当的时间管理（下一位客人前2小时缓冲）
  • 有问题时立即警报
  • 通过照片验证的责任制

5.8.2 手动任务创建

何时创建手动任务：

手动任务用于非退房清洁场景： - 深度清洁 - 每月或每季度的详细清洁 - 入住前准备 - 新房产的第一位客人 - 维护清洁 - 维修或翻新后 - 特殊活动 - 派对后清理 - 季节性任务 - 擦窗、地毯清洁、室外维护 - 紧急清洁 - 客人提前离开并留下脏乱

基本信息： 1. 点击团队标签中的”创建任务”按钮 2. 房产自动填充（当前房源） 3. 任务类型自动设置为”手动”

任务详情： 1. 优先级（必需） - 低 - 可以等待3-5天 - 正常 - 应在2天内完成（默认） - 高 - 需要24小时内 - 紧急 - 需要当天

  1. 截止时间（必需）
    • 日期选择器 - 选择目标完成日期
    • 时间选择器 - 选择目标完成时间
    • 默认：如果未指定则为今天下午5:00
    • 系统验证截止时间在未来
  2. 预估持续时间（必需）
    • 下拉菜单：1小时、1.5小时、2小时、2.5小时、3小时、4小时
    • 默认：2小时（标准周转）
    • 用于清洁人员调度和分配
  3. 备注（可选）
    • 自由文本字段（最多500字符）
    • 描述所需的具体任务
    • 示例：“深度清洁厨房，重点是烤箱和冰箱。客人报告地板粘腻。”
    • 示例：“外窗清洗，所有窗户从地面可触及。”

第3步：分配

选项 A：自动分配（推荐） - 留空分配 - 系统使用智能算法（见 5.8.3）找到最佳清洁人员 - 考虑：可用性、工作量、评分、距离

选项 B：明确分配 - 从下拉菜单选择特定清洁人员 - 仅显示分配给此房产的清洁人员 - 系统验证清洁人员在截止时间可用 - 如果不可用，显示警告并建议替代方案

审核与创建： 1. 审核任务摘要： - 房产名称 - 优先级徽章（颜色编码） - 截止时间（日期和时间） - 预估持续时间 - 分配（自动或特定清洁人员名称） - 备注预览 2. 点击”创建任务”按钮 3. 任务创建，清洁人员立即收到通知 4. 重定向到任务详情视图

5.8.4 清洁人员通知与确认

清洁人员操作：

  2. 确认 - 点击”我来做”确认（可选但推荐）

5.8.5 任务生命周期与状态

任务状态流程：

状态定义：

  1. 已创建（0-2秒）
    • 任务刚创建，分配进行中
    • 清洁人员尚不可见
  2. 待处理（直到清洁人员开始）
    • 已分配给清洁人员
    • 清洁人员已通知
    • 等待清洁人员开始
    • 房东仍可修改或取消
  3. 进行中（清洁人员已开始）
    • 清洁人员点击了”开始任务”
    • 计时器运行中（跟踪持续时间）
    • 房东可以查看实时状态
    • 一旦开始无法重新分配
  4. 已完成（清洁人员已提交）
    • 清洁人员上传了照片
    • 标记了所有清单项目完成
    • 已提交供房东审核
    • 房东现在可以评价清洁人员
  5. 已评分（最终状态）
    • 房东评价了清洁人员（1-5星）
    • 任务关闭
    • 数据保存用于分析

5.8.7 任务绩效与分析 [仅房东]

关键指标：

  1. 平均完成时间
    • 清洁人员用时 vs 预估
    • 按房产大小分解
    • 趋势（改进还是变慢？）
  2. 准时率
    • 截止时间前完成的任务百分比
    • 按清洁人员（识别可靠性）
    • 按优先级（紧急任务表现）
  3. 质量评分
    • 所有任务的平均评分
    • 按清洁人员（谁最好？）
    • 按房产（哪个最难清洁？）
  4. 任务量
    • 每月创建的总任务
    • 自动 vs 手动分解
    • 高峰日/时间
  5. 清洁人员利用率
    • 每个清洁人员的工作小时
    • 分配均匀还是不平衡？
    • 识别超负荷/闲置的清洁人员

每种房间类型（卧室、浴室、厨房等）的详细任务列表，清洁人员遵循以确保一致质量。

    • 检查过期食品
  4. 客厅 🛋️

    • 擦拭家具和电子设备灰尘
    • 整理靠垫、折叠毯子
    • 清空垃圾
    • 检查遥控器电池

    • 清空垃圾/烟灰缸
    • 检查危险
    • 浇花（如有要求）

清单功能：
